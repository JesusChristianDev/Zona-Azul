# Iconos PWA necesarios

Para completar la configuración PWA, necesitas crear los siguientes iconos:

1. **icon-192x192.png** - 192x192 píxeles
2. **icon-512x512.png** - 512x512 píxeles
3. **favicon.ico** - 32x32 píxeles (opcional pero recomendado)

Puedes generar estos iconos usando:
- [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- O cualquier herramienta de diseño que genere PNG

El icono base puede ser un logo de HealthyBox con las iniciales "HB" en verde (#059669).

También puedes usar el archivo `public/icon.svg` como base para generar los iconos.
